# Backend Tech Stack

## Selected Technologies

### Web Framework
- **Express 5.x** - Minimal and flexible Node.js web application framework

### Database & ORM
- **Drizzle ORM** - TypeScript-first ORM for type-safe database interactions
- **PostgreSQL Driver (pg)** - Native PostgreSQL client for Node.js

### Data Validation
- **Zod** - TypeScript-first schema validation library

### Security
- **Helmet** - Secure Express.js applications by setting various HTTP headers
- **Better Auth** - Authentication library for modern web applications

### Logging
- **Pino** - High-performance JSON logger for Node.js
